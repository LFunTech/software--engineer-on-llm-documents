## ADDED Requirements
### Requirement: Implementation cookbook
The repository SHALL provide a vibe-coding cookbook for implementing frontend and backend work using AI tools with existing project conventions.

#### Scenario: Frontend flow guidance
- **WHEN** a teammate opens `docs/vc-implementation-cookbook.md`
- **THEN** they find prompts and step-by-step flows for React/SPA component/page creation, styling integration, and state/data wiring, including test/story guidance

#### Scenario: Backend flow guidance
- **WHEN** reading the cookbook
- **THEN** it includes prompts and steps for API routes/handlers, validation, logging, error handling, and data access patterns aligned to project conventions

#### Scenario: Minimal patterns and templates
- **WHEN** following the cookbook
- **THEN** it supplies minimal code/prompt templates that can be adapted, keeping changes small and reviewable
